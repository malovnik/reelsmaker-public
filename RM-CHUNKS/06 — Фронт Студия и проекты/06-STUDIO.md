## Суть этапа

Главная страница приложения = **Студия**. Это grid проектов, как в YouTube Studio или в CapCut/DaVinci home-screen: превью-карточки, статус, дата, быстрые действия. Одна цель — «вошёл и сразу видишь свои проекты».

Функционал владельца (пересказано дословно):
- Сетка последних проектов с контекстным меню: переименовать, удалить навсегда, удалить из сетки (soft-delete), открыть в Finder/Проводнике.
- Кнопка «Новый проект» → модалка: **сразу закидываем видео**, выбираем «настройки с нуля» / «скопировать из предыдущего» / «выбрать из списка прошлых».
- Автосохранение каждые 10 с + ручное.
- Поиск/фильтры/сортировка сверху.

**Режим работы:** Sequential. 6 чанков. Каждый — одна видимая фича.

## Выход

- `apps/frontend/src/routes/index.tsx` — главная (Студия).
- `apps/frontend/src/features/studio/` — все компоненты студии.
- `apps/frontend/src/features/projects/` — CRUD хуки и UI.
- Работающие сценарии: создать проект → закинуть видео → увидеть превью в сетке → переименовать → удалить.

## Принципы UI (выжимка из референсов)

- **Dark mode по умолчанию**, яркие thumbnail-превью — главный визуальный акцент.
- Скругления умеренные (8–12 px), не «пузыри».
- Hover-эффекты через scale 1.02 + shadow-lift, не overkill.
- Контекстное меню — через Radix DropdownMenu (без shadcn), со своим стилем из design-system.
- Кнопка «Новый проект» — primary CTA, акцентный цвет (не тусклый).
