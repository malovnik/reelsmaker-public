## Микрозадачи

### 41.1 Триггер

Кнопка primary «Новый проект» в StudioHeader (REFACTR-43) или большая кнопка в Empty state.

### 41.2 Layout модалки

Одна страница (не wizard с шагами):

```
┌─────────────────────────────────────┐
│  Новый проект                    ✕  │
├─────────────────────────────────────┤
│                                     │
│  [ Drop-zone / File select ]        │
│   Закинь видео или выбери файл     │
│   mp4, mov, mkv до 10 ГБ            │
│                                     │
│  [ ~~~ прогресс upload ~~~ ]        │
│                                     │
├─────────────────────────────────────┤
│  Название проекта                   │
│  [ Input: автозаполняется из имени] │
├─────────────────────────────────────┤
│  Настройки                          │
│  ◉ С нуля (профиль Viral 2026)      │
│  ◯ Скопировать из предыдущего       │
│  ◯ Выбрать из списка  [ Project ▾ ] │
├─────────────────────────────────────┤
│             [ Отмена ] [ Создать ]  │
└─────────────────────────────────────┘
```

### 41.3 Drop-zone

`src/features/studio/NewProjectModal/DropZone.tsx`:

- react-dropzone (Context7 актуальная версия) или нативный drag-events.
- States: idle, dragover, uploading, uploaded, error.
- Визуально: dashed border (в idle), solid accent (dragover), progress bar (uploading), checkmark + имя файла (uploaded).

### 41.4 Upload

`POST /api/uploads` (streaming, если > 500 МБ — chunked). Progress — через XHR progress events или `fetch` с `ReadableStream`.

Backend endpoint — уже существует или добавить.

### 41.5 Auto-name

При uploaded — заполнить Input «Название проекта» из имени файла без extension. Пользователь может отредактировать.

### 41.6 Выбор настроек

- Radio: «С нуля» / «Скопировать из предыдущего» / «Выбрать из списка».
- При «Скопировать из предыдущего» — подсказка: «Будет использован последний проект: *Title*, *2 дня назад*».
- При «Выбрать из списка» — появляется Select с проектами (API `GET /api/projects?for_copy=true`, REFACTR-17).

### 41.7 Создание

Кнопка «Создать» активна только когда:
- Файл загружен.
- Название не пустое.
- Выбор настроек сделан.

По клику: `POST /api/projects` с `{name, video_upload_id, copy_settings_from_project_id?}`. Ответ → navigate в Workbench.
