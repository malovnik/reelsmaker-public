## Микрозадачи

### 42.1 Быстрый вариант («Последний»)

В модалке NewProject — radio «Скопировать из предыдущего»:

- API: `GET /api/projects?for_copy=true&limit=1&sort=last_saved_at_desc` — получить последний.
- Показать inline: «Будет использован: *Reels Алматы 12.04*, сохранён 2 часа назад». Кнопка-ссылка «Выбрать другой».

### 42.2 Полный picker

Клик «Выбрать другой» (или выбор radio «Выбрать из списка») → inline-раскрытие:

```
[ Поиск по названию           ]
┌──────────┬─────┬──────────┐
│ 🖼 Thumb │ Name │ last_saved │
├──────────┼─────┼──────────┤
│ ...      │ ... │ 2 ч назад │
│ ...      │ ... │ 1 день    │
│ ...      │ ... │ 3 дня     │
│ ...      │ ... │ 2 недели  │
└──────────┴─────┴──────────┘
  показано 20 из 47.  [ Ещё ]
```

### 42.3 Реализация

`src/features/studio/NewProjectModal/SettingsPicker.tsx`:

- Query: `GET /api/projects?for_copy=true&limit=20` → TanStack Query.
- Поиск (debounced): фильтр по name client-side (для 20 проектов) или server-side с query-param.
- Select: click на row → подсвечивается, сохраняется в state модалки.

### 42.4 Применение

При «Создать» проект в payload POST передаётся `copy_settings_from_project_id`. Backend делает copy (REFACTR-17).

### 42.5 После создания

Toast: «Создан проект *X*, скопированы настройки из *Y*.» На Workbench уже видны настройки.
