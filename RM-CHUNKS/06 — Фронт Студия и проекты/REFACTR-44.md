## Микрозадачи

### 44.1 EmptyState компонент

`src/features/studio/EmptyState.tsx`:

```
      ┌─────────────────┐
      │                 │
      │   [ иконка ]    │
      │                 │
      └─────────────────┘

    Тут будут твои видео

  Закинь первое — студия нарежет
  его на рилсы и шортсы.

  [ + Новый проект ]    [ ? Как это работает ]
```

- Иконка: кинокамера или monitor-play из lucide, размер 48 px, цвет accent-muted.
- Title: text-xl, weight 600.
- Subtitle: text-md text-secondary, 1-2 строки.
- Primary кнопка «Новый проект» + ghost кнопка «Как это работает» (→ модалка онбординга или внешний help).

### 44.2 Условие отрисовки

В ProjectGrid:
```tsx
if (!isLoading && projects.length === 0) {
  return <EmptyState />;
}
```

### 44.3 Онбординг-модалка (опционально)

Короткая модалка на 3-4 шага (скриншоты или картинки):
- Шаг 1: Закинь видео.
- Шаг 2: Студия сама делает транскрипцию и предложит идеи рилсов.
- Шаг 3: Ты одобряешь / отклоняешь / регенерируешь.
- Шаг 4: Рендер и скачивание.

Показывается при первом клике «Как это работает». Persist «видел онбординг» в localStorage → не мешает повторным заходам, но доступен через Help-меню.
