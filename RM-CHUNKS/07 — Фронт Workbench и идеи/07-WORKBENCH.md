## Суть этапа

Страница одного проекта = **Workbench**. Это где вся работа: загруженное видео, прогресс стадий pipeline, список идей рилсов с approve/reject/regenerate, управление рендером.

Главные сценарии владельца:
1. «Пришёл, видео уже загружено, вижу все стадии: транскрипция → удаление тишины → идеи → склейка → рендер».
2. «Хочу перезапустить процесс заново с шага (например, с транскрибации, с выделения рилсов)» — список стадий с кнопкой «Начать заново отсюда».
3. «Сначала система генерирует идеи рилсов — описание, текст, хук, таймкоды — я их approve/reject/regenerate до рендера».
4. «Либо жму одну кнопку — полностью автоматический режим, без моих approve-шагов».
5. «Regenerate с опциональным prompt» — модалка с инструкцией «что пересмотреть».

**Режим работы:** Sequential. 6 чанков.

## Вход

- Grid проектов из Этапа 06 → клик → Workbench.
- API идей (approve/reject/regenerate) из Этапа 02 (REFACTR-19, REFACTR-20).
- API перезапуска с шага из Этапа 02 (REFACTR-16).
- SSE endpoint прогресса (существует в текущем backend).
- **Skill `frontend-design` обязателен в каждом чанке.**

## Выход

- `apps/frontend/src/routes/jobs/$jobId.tsx` — Workbench.
- `apps/frontend/src/features/workbench/` — компоненты.
- `apps/frontend/src/features/ideas/` — управление идеями (GridIdeas, IdeaCard, RegenerateModal).
- Работающий сценарий: открыть проект → дождаться идей → approve 3 / reject 2 / regenerate 1 → запустить рендер → увидеть результат.

## GATE-чекпоинт этапа

- [ ] Workbench открывается из Студии по клику на карточку проекта.
- [ ] Pipeline-timeline показывает текущую стадию live (через SSE).
- [ ] «Начать заново с шага X» — работает, не теряет настройки проекта.
- [ ] В режиме «Полная автоматика» pipeline идёт без остановок и доходит до готовых клипов.
- [ ] В режиме «Пошагово» — pipeline останавливается на идеях, ждёт approve.
- [ ] Approve/Reject/Regenerate — работают, UI реагирует мгновенно (optimistic updates), сервер подтверждает.
- [ ] Regenerate без prompt использует стандартный промпт (видно в логах бэка).
- [ ] Готовые клипы можно скачать + открыть папку в Finder.
