## Микрозадачи

### 48.1 Layout карточки

```
┌──────────────────────────────────────────┐
│ ● Pending           Идея #3     regen 0  │
├──────────────────────────────────────────┤
│ Заголовок идеи (text-lg 600)             │
│ ────────────────────────────────────      │
│ Описание: 2-3 строки text-sm secondary   │
│                                          │
│ Хук (первые 3-5 сек):                     │
│   "Один из главных мифов о... "          │
│                                          │
│ Текст рилса:                              │
│   "..." (раскрывается если >180 символов)│
│                                          │
│ Таймкоды:                                 │
│   0:00 → 12:34-12:42  (8s)               │
│   0:08 → 34:12-34:28  (16s)              │
│   0:24 → 41:55-42:07  (12s)              │
├──────────────────────────────────────────┤
│  [ ✗ Отклонить ]  [ 🔄 Регенер ]  [ ✓ Approve ] │
└──────────────────────────────────────────┘
```

### 48.2 Grid

Responsive 1-2 колонки:
- На широких экранах (>1440) — 2 колонки (более плотно).
- На узких — 1 колонка (читать удобно).

Загрузка: SSE `ideas_ready` → invalidate Query → идеи появляются.

### 48.3 Preview таймкодов

Клик на таймкод (`12:34-12:42`) → video player перескакивает на этот момент и проигрывает фрагмент. Полезно для «а что там на самом деле сказано».

### 48.4 Статусы визуально

- pending: серый dot.
- approved: зелёный dot.
- rejected: красный dot.
- regenerating: желтый spinner + надпись «Регенерируется…».

Карточки approved слегка приглушённые (opacity 0.7), rejected — ещё сильнее. Regenerating — спиннер поверх.

### 48.5 Фильтры

Над grid — chip-фильтр: Все / Pending / Approved / Rejected. Счётчики.

Клик — фильтр. URL-param.

### 48.6 Bulk actions

Если нажать Shift + Click на несколько карточек — multi-select. Bulk toolbar:
- Approve selected.
- Reject selected.
- Regenerate selected.

(опционально, если успеваем — иначе в future).
