# Этап 08: Фронт — настройки и Cmd+K

> Статус: ⬜ Не начат
> Родитель: [[PIPELINE-НАВИГАТОР]]
> Проект: **videomaker-рефакторинг**

## Суть этапа

Настройки — самое больное место текущего сервиса. Горизонтальная прокрутка в `/settings/subtitles`, post-production как свалка, brand-kit рядом с prompts без логики группировки. Владелец цитата: «омерзительно-омерзительно-омерзительно».

Этап переизобретает настройки полностью:

1. **Новая навигация**: вертикальная (не горизонтальная), группы по смыслу (не по историческому накоплению).
2. **Каждая настроечная секция**: адаптивный layout, mobile-first, никакого horizontal scroll.
3. **Режимы**: Простой (показывает только ключевые опции) / Экспертный (показывает всё, сворачиваемые аккордеоны).
4. **Cmd+K Command Palette**: глобальный поиск по проектам, настройкам, действиям — с клавиатурной навигацией.

**Режим работы:** Sequential. 7 чанков.

## Подэтапы (REFACTR-51..REFACTR-57)

- **REFACTR-51** — Новая IA настроек: группы (Запись/Обработка/Визуал/LLM/Интеграции/Девайс) ⬜
- **REFACTR-52** — Settings/Subtitles: адаптивный редактор (без гор.прокрутки), preview рядом ⬜
- **REFACTR-53** — Settings/Post-production: группы параметров (звук/цвет/переходы/субтитры/эффекты) ⬜
- **REFACTR-54** — Settings/Models + Prompts: единый редактор промптов с версионностью ⬜
- **REFACTR-55** — Settings/Brand + Performance + Profiles + Connections ⬜
- **REFACTR-56** — Режимы Simple/Expert (toggle + persist) ⬜
- **REFACTR-57** — Cmd+K Command Palette (проекты + настройки + действия) ⬜

## Вход

- Карта настроек из Этапа 00 (REFACTR-02).
- Дизайн-система (Этап 05).
- Studio+Workbench (Этапы 06, 07) для интеграции Cmd+K.
- **Skill `frontend-design` обязателен в каждом чанке.**

## Выход

- `apps/frontend/src/routes/settings/` — все маршруты настроек.
- `apps/frontend/src/features/settings/` — компоненты (SettingsNav, группы).
- `apps/frontend/src/features/command-palette/` — Cmd+K.
- Все настройки сохраняются через автосейв (debounce 500 мс, не путать с 10-сек автосейвом проекта).

## Принципы IA настроек

| Группа | Что внутри |
|--------|-----------|
| Запись / Транскрипция | провайдер (MLX/Deepgram), модель, фильтры шума, VAD |
| Обработка | удаление тишины, филлеры, multi-pass LLM, narrative pipeline |
| Визуал | B-roll, переходы, Ken Burns, цвет, LUT, эффекты |
| Субтитры | стиль, шрифты, таймкоды, позиционирование |
| LLM | модели (Gemini/Claude/GPT), промпты, температуры |
| Интеграции | API-ключи (Deepgram, Gemini, Anthropic), выходные папки |
| Девайс | темы, режим Simple/Expert, горячие клавиши |
