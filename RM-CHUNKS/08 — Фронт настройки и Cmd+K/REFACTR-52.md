### 52.2 Preview

SubtitlePreview компонент — уже существует в legacy, переносим, адаптируем:
- Рендерит mini-canvas с одним кадром + субтитры по текущим настройкам.
- Обновляется live по изменению настроек (debounce 100 мс).
- Play/pause — для анимированных субтитров (karaoke-style, если настроено).

### 52.3 Форма

`react-hook-form` или контролируемый state с `useMutation` для PUT.

Все поля autosave (debounce 500 мс). Индикатор «Сохранено» в хедере.

### 52.4 Horizontal scroll CHECK

Явный тест при разработке:
- Viewport 1024 px → preview уходит вниз, настройки на всю ширину, без h-scroll.
- Viewport 768 px → то же.
- Viewport 1920 px → two-panel.

Devtools responsive mode — обязательный тест.

### 52.5 Ключевые настройки

Выписать все поля из `SubtitleStyleEditor.tsx` legacy, разложить в группы:

- **Шрифт и размер**: семья, размер, жирность.
- **Цвета**: текст, обводка, фон-плашка (опционально).
- **Позиция**: top/center/bottom + custom offset.
- **Переносы и длина**: max chars per line, max lines.
- **Вид**: burn-in / SRT.
- **Эффекты**: karaoke-style, highlight words (опционально).
