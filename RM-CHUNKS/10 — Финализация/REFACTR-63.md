## Микрозадачи

### 63.1 Сценарий

1. Открыть existing проект (`reference-project`), изменить настройки (LLM = Claude Sonnet, Subtitles font = 32px, Post-production LUT = Cinema).
2. Подождать autosave (10 с или ручной).
3. В Студии клик «Новый проект».
4. Upload тестового видео.
5. Выбрать «Скопировать из предыдущего» — подставляется `reference-project`.
6. Клик «Создать».
7. В Workbench → Settings → проверить каждое значение: совпадает с `reference-project`.

### 63.2 Автоматизация

Сравнение через API:

```bash
curl /api/projects/REFERENCE_ID/settings > ref.json
curl /api/projects/NEW_ID/settings > new.json
diff ref.json new.json  # должно быть пусто, кроме project-specific полей
```

Project-specific поля (не копируются): `name`, `source_video_path`, `created_at`, `id`.

Остальное — должно совпадать.

### 63.3 Альтернативный сценарий: picker

1. Создать второй «новый проект».
2. В модалке — «Выбрать из списка» → picker открывается.
3. Поиск по названию «reference».
4. Выбрать `reference-project` → «Создать».
5. Проверить настройки — те же.
