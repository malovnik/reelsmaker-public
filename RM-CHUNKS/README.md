## Зачем этот план

Текущий videomaker — работает, но перегружен:

1. **Фронт жрёт 12 ГБ RAM в dev** (`NODE_OPTIONS=--max-old-space-size=12288` в `package.json`) — OOM при забитой памяти.
2. **Дизайн — каша:** смешанные тёмные/светлые шрифты, где-то читаемо, где-то нет, нет цельной темы, нет persist.
3. **Настройки хаотичны:** субтитры с горизонтальной прокруткой без адаптива, post-production без структуры, все параметры в одной куче.
4. **Нет «Студии» на главной:** проекты, их переименование, удаление навсегда/из сетки, открытие в Finder, кнопка «Новый проект» — требуют переизобретения.
5. **Нет автосохранения** настроек проекта + «перезапустить с шага X».
6. **Нет потока идей рилсов** с approve/reject/regenerate до рендера.
7. **Поиск Cmd+K не работает**.
8. **Pro-профиль болтается в коде** — нужен только Viral 2026 + legacy chapter bot.
9. **Видеодвижок не оптимизирован под M5 Pro** (VideoToolbox hardware encode не проверен и не зафиксирован).

Цель плана — переизобрести сервис так, чтобы он выглядел как зрелый продакшн-продукт 2026 года (YouTube/Instagram-динамика, тёмная/светлая темы, persist), при этом не потерял ни одной рабочей фичи pipeline.

## Принцип нумерации для лупера

**Единый префикс для всех PROD-чанков:** `REFACTR-NN.md`, где NN — сквозная нумерация от 00 до 66.

Лупер `/Users/malovnik/Documents/malovnik-obsidian/🤖 ИИ/Софт/looper-tmux` находит файлы по префиксу `REFACTR-*.md` и идёт по ним в порядке возрастания номера. **Переходит к следующему чанку только после GATE-чекпоинта.**

L1-файлы этапов (`00-АУДИТ.md`, `01-РЕШЕНИЯ.md` и т.п.) не имеют префикса `REFACTR` — это оглавления этапов, лупер их читает как контекст, но не исполняет.

## Деливери каждого чанка

По образцу Мусатова, **1 чанк = 1 артефакт в коде или документация в `docs/`**. Никаких TODO/FIXME/mocks. Production-ready с первой строки.

## Ключевые ограничения (НЕ нарушать)

- **Удалить полностью**: pro-профиль (код, UI, storage).
- **Оставить**: Viral 2026 (основной), legacy chapter bot.
- **Темы**: тёмная (основная) + светлая, CSS variables, persist в localStorage + копия на бэке.
- **Без логинов** (локальное приложение, один пользователь).
- **Автосохранение**: дебаунс 10 с + ручное сохранение.
- **Скиллы обязательны**: `frontend-design` для любого UI-чанка в этапах 05–08, `role-factory` для генерации ролей специалистов (backend/frontend/security).
- **Запуск через `run.sh`** с preflight-cleanup, проверкой зависимостей и graceful shutdown.
- **Рендер** оптимизирован под VideoToolbox на M5 Pro, fallback на software — обязателен.
- **ТАБУ в контенте** (политика/военные/религия/политики/крипта) — не трогаем, внутреннего влияния нет.
